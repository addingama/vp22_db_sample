# Contoh Aplikasi Koneksi ke Database

Aplikasi ini merupakan panduan *step by step* untuk menerapkan koneksi database MySQL menggunakan Visual Studio

### Panduan Clone

1. Buka Visual Studio
2. Pilih Clone Project
3. Masukkan `https://github.com/Rohmatulislam/DBConnectionSample.git` sebagai Git URL
4. Lakukan import `menjual.sql` ke database
5. Ubah `DBConnectionModule.module` menggunakan `user_id` dan `password` sesuai dengan yang anda gunakan
